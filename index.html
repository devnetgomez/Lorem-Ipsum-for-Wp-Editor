Lorem Ipsum for WP Editor

Now you can test you website adding Lorem Ipsum text to your posts using this WordPress plugin

Download .zip Download .tar.gz View on GitHub

Installing

It's so easy to use in your website! You just need download and activate :)

How it works

When you are creating/editing the post, you can choose how many paragraphs you want to add on your text. After choose, the Lorem Ipsum text will be added after the existent text.
